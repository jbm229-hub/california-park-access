California Park Accessibility Analysis

This project analyzes accessibility to California state and national parks using traffic-based drivability metrics, pedestrian isochrones, and Census demographic data. All maps were produced using Python, GeoPandas, OSMnx, and Folium.

Methods Summary

Drivability scores are calculated as an inverse-distance weighted average of Annual Average Daily Traffic (AADT) from nearby road segments. Walking accessibility is estimated using OpenStreetMap pedestrian networks, with isochrones generated at 15â€“60 minute intervals. Census tract demographics are spatially intersected and population-weighted within each isochrone.
